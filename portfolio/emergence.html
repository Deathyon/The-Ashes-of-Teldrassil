Emergence


Emergence is a first person, turret defense style game developed in Unreal Engine 4. I served as the Technical Lead for this project and managed a team of 5 programmers and all technical aspects of the project.


As the Technical Lead for Emergence I setup version control for the entire team using perforce and hamachi. I also authored documentation detailing the technical specifications, goals, and challenges of the project. I managed and delegated tasks for the programming team with Trello and reviewed and approved tasks prior to releases.



For this project I scripted the player and level controllers using Blueprints within the Unreal Engine Editor. I also contribitued to the scripting of the UI/HUD, Enemy AI, and the designing of levels, mechanics, and player/enemy abilities.
