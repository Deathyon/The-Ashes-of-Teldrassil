Emergence


Astroid is a cutting edge proton radiation therapy planning system. It is the only FDA aproved radiation therapy planning system to take advantage of microservices and cloud computing to run the extremely complex calculations required for treatment planning. This empowers users to work from virtually anywhere with an internet connection.


As a software developer at .decimal, I simultaneously designed, programmed, and implemented UI/UX elements for Astroid. On a daily basis, I wrote, reviewed, and maintained C++ code. I worked inside of an agile environment utilzing tools such as JIRA, Visual Studio, Git, and Jenkins, among others.


Astroid consists of several applications working together to deliver the most advanced proton therapy planning system to date. On the backend, the Thinknode app handles requests for the Planning app and delivers calculation results as well as maintains an immutable database of patient records. The Planning app is where users will upload, search for, and create patient data and treatment plans. From the Planning app, users will then export published data to the MAIA app where it is QA'd prior to the patient recieving treatment.
